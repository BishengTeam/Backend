## 题库模块

> 全部接口已完成单元测试、PostgreSQL 数据库集成测试和 HTTP 冒烟测试。

✅ GET /api/quiz/categories — 题库分类树（公开，递归构建多级子分类，循环引用保护）
✅ GET /api/quiz/questions — 题目列表（公开，支持 category_id / question_type 筛选 + 分页，不泄露 correct_answer）
✅ POST /api/quiz/submit — 答题提交 + 即时判分（需登录，支持 single_choice / multiple_choice / judge 三种题型，中文判断题答案「对/错」归一化为 TRUE/FALSE，答错自动入错题本）
✅ GET /api/quiz/wrong-book — 错题本列表（需登录，用户隔离 + 分页，返回题目详情不含答案）
✅ POST /api/quiz/wrong-book — 错题本添加（需登录，无答题记录时自动创建，幂等）
✅ DELETE /api/quiz/wrong-book/{id} — 错题本移除（需登录，将 is_wrong 置为 false，非物理删除）
✅ GET /api/quiz/collections — 收藏列表（需登录，用户隔离 + 分页，返回题目详情不含答案）
✅ POST /api/quiz/collections — 收藏添加（需登录，无答题记录时自动创建，幂等）
✅ DELETE /api/quiz/collections/{id} — 收藏移除（需登录，将 is_collected 置为 false，非物理删除）
✅ GET /api/quiz/checkin — 打卡状态查询（需登录，返回当天打卡状态、连续天数、最后一次打卡日期）
✅ POST /api/quiz/checkin — 打卡签到（需登录，重复签到幂等返回已有记录，连续天数基于昨天签到记录自动计算）
