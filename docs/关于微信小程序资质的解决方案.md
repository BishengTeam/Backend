# 关于微信小程序资质的解决方案

> 日期：2026-05-16

---

## 一、问题与背景

公司方计划通过微信小程序开展两项业务：
- **考试券售卖**（职业认证报名，如 H3C NE、深信服 CSE、NISP 等）
- **网课交付**（在线视频课程）

微信小程序对教育培训类目设有前置资质要求——须提交《民办学校办学许可证》。公司方目前未持有该证，直接申请「教育培训」类目将无法通过审核。

经梳理，以下列出三条可选路径，供公司方决策参考。

---

## 二、三条可选路径

| | 方案A | 方案B（推荐） | 方案C |
|------|------|------|------|
| **思路** | 申请办学许可证，正规入驻 | 与持证培训机构合作，业务分离 | 技术手段绕过类目审核 |
| **上线时间** | 3-6 个月 | 2-4 周 | 1-2 周 |
| **前期投入** | 办学场地+人员+行政费用 | 合作谈判+少量技术对接 | 纯技术开发 |
| **持续成本** | 场地租金+人员薪酬 | 合作分润或固定服务费 | 服务器+域名维护 |
| **小程序类目** | 教育培训（全功能） | 教育信息服务 | 教育信息服务 |
| **课程承载方式** | 小程序内直接承载 | 合作方平台（跳转访问） | 自建外部站点（跳转访问） |
| **营收归属** | 100% | 扣除合作分润 | 100% |
| **品牌独立性** | 完全自有 | 需标注合作方品牌 | 完全自有 |
| **合规程度** | 完全合规 | 合规 | 存在风险 |
| **下架风险** | 无 | 低 | 中-高 |

## 三、推荐方案：与持证培训机构合作（方案B）

### 3.1 运作模式

核心逻辑是业务分离——公司方与持证机构分工协作：

- **公司方（小程序端）**：负责认证类型展示、考试资讯、报名收费——此类信息服务无需办学许可证
- **合作方（持证机构）**：负责课程制作、教学交付、在线答疑——此类教学服务须持有办学许可证

用户于小程序端完成报名与支付，点击「开始学习」后跳转至合作方教学平台上课。用户端体验流畅，教学服务的法律主体为持证合作机构。

### 3.2 合规依据

教育信息服务类目（无需办学许可证）允许：信息展示、报名服务、费用代收。教学交付由持有办学许可证的合作方独立完成，公司方经营范围与平台类目许可均不超出边界。

关键原则：**审核期与上线后，小程序内展示内容完全一致**，课程页面明确标注「教学服务由 XX 培训学校提供」。

### 3.3 落地步骤

1. **筛选合作方**：寻找一家持有办学许可证、具备在线教学能力的培训机构
2. **签订合作协议**：明确分工（公司方负责招生推广，合作方负责教学交付）、分润比例、退出机制
3. **技术对接**：小程序与合作方教学平台打通（用户支付后自动开通课程权限）
4. **提交审核**：以「教育信息服务」类目提交微信审核，课程页面如实标注教学提供方

### 3.4 合作方筛选标准

- 持有有效的《民办学校办学许可证》
- 办学范围涵盖职业资格培训或信息技术培训
- 具备成熟的在线教学平台（录播+直播+题库）
- 经营合规，无行政处罚记录（可通过信用中国查询）

### 3.5 成本与周期

- **上线周期**：筛选合作方 1-2 周 + 协议谈判 1 周 + 技术对接 1 周 ≈ 2-4 周
- **成本构成**：主要为合作分润（按学员人数或课程套餐结算），无额外前置投入

## 四、长期方案：申请办学许可证（方案A）

### 4.1 适用条件

若公司方计划长期深耕教育培训行业，建议同步启动办学许可证申请——方案B 先行上线开展业务运营，方案A 同步推进，获取自有资质。

### 4.2 前置条件

| 条件 | 说明 |
|------|------|
| 办学场地 | 符合消防和面积要求（各地标准不同） |
| 管理人员 | 校长任职资格 + 教师资格证 |
| 注册资金 | 各地 10-100 万不等，需实缴 |
| 办学章程 | 可参照模板编制 |
| 消防验收 | 住建部门出具 |

### 4.3 获证后收益

- 小程序类目可切换为「教育培训」，课程直接在小程序内播放
- 不再依赖合作方，营收 100% 归属
- 可开设微信小店教育培训店铺，扩展销售渠道
- 从根本上消除类目合规风险

### 4.4 周期

从准备材料至获批，通常需 3-6 个月。

---

## 五、紧急备选：快速上线方案（方案C）

### 5.1 运作方式

选择「教育信息服务」类目（无需办学许可证），小程序内仅展示认证介绍与报名支付功能，课程内容部署于外部 H5 站点，通过小程序 web-view 加载。

审核期间，课程入口对审核人员不可见；审核通过后，正式版向用户开放。

### 5.2 ⚠️ 风险提示

本方案上线速度最快，但存在以下合规风险，公司方决策前须充分知悉：

| 风险 | 说明 |
|------|------|
| **平台下架** | 审核期间隐藏真实功能，违反微信小程序运营规范。一经抽检或举报，可能面临下架、主体封禁、结算资金冻结 |
| **虚假宣传** | 小程序展示的「报名服务」与实际交付的「课程培训」存在描述不一致，可能构成虚假宣传 |
| **超范围经营** | 实际从事教育培训业务但营业执照未覆盖，存在行政处罚风险 |
| **行政许可缺失** | 无办学许可证开展教学业务，法律责任不因「平台审核通过」而免除 |

> 方案C 以技术手段换取上线速度，合规风险不可忽视。建议仅作为方案B 短期无法落实时的过渡方案，且须在方案B 就位后立即切换。
